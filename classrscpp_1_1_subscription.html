Detailed Description

A rscpp::Subscription represents a one-to-one lifecycle of a rscpp::Subscriber subscribing to a rscpp::Publisher.

It can only be used once by a single rscpp::Subscriber.

It is used to both signal desire for data and cancel demand (and allow resource cleanup).
