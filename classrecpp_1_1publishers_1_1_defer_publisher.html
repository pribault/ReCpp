template<typename T , typename P >
void recpp::publishers::DeferPublisher< T, P >::subscribe ( rscpp::Subscriber< T > &  subscriber)
overridevirtual

Request rscpp::Publisher to start streaming data.

This is a "factory method" and can be called multiple times, each time starting a new rscpp::Subscription.

Each rscpp::Subscription will work for only a single rscpp::Subscriber.

A rscpp::Subscriber should only subscribe once to a single rscpp::Publisher.

If the rscpp::Publisher rejects the subscription attempt or otherwise fails it will signal the error via rscpp::Subscriber::onError(const std::exception_ptr &).

Parameters
subscriberThe rscpp::Subscriber that will consume signals from this rscpp::Publisher
